## 📌 배경

데이터 시각화 대시보드는 개인 데이터, 소셜 트렌드, 기술 통계 등 다양한 영역에서 인사이트를 도출하는 핵심 도구입니다.
본 아이디어 모음은 3D 글로브/지도 기반 시각화를 제외하고, 차트·그래프·히트맵 등 다양한 시각화 기법을 활용하는 프로젝트를 제안합니다.
기존 sigco3111 저장소의 대시보드/시각화 프로젝트와 중복되지 않는 아이디어로 구성했습니다.

### 참고: 기존 관련 저장소
- `co2-board` — 국가별 CO₂ 배출량 비교 분석 대시보드
- `earthquake` — 전 세계 실시간 지진 추적 대시보드
- `commodity-dashboard` — 실시간 원자재 시세 모니터링
- `world-bank-board` — 세계은행 데이터 AI 비교 분석
- `github-board` — GitHub 사용자 프로필 대시보드
- `youtube-board` — YouTube 채널 분석 대시보드
- `dev-trend-oracle` — 개발 트렌드 오라클 대시보드
- `algo-visualizer` — 알고리즘 & 자료구조 시각화
- `relationship-visualizer` — AI 창작물 인물 관계도 시각화
- `repo-dna-analyzer` — 코드 스타일 지문 분석기
- `git-constellation` — GitHub 커밋 히스토리 별자리 시각화
