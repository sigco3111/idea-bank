## 📖 이 저장소는?

아이디어를 체계적으로 수집하고 관리하는 공간입니다.  
웹 개발, 게임, 시각화, 자동화 등 다양한 분야의 프로젝트 아이디어를 브레인스토밍하고 정리합니다.

모든 아이디어는 **GitHub + Vercel 배포 가능**, **무료 기술 스택 기준**, **기존 저장소와 중복 없음**을 원칙으로 합니다.
