# 🔬 과학 & 시뮬레이션 프로젝트 아이디어 모음
> Canvas, WebGL, 물리엔진 기반 GitHub + Vercel 배포 가능 프로젝트 아이디어  
> 생성일: 2026-04-28 | 작성: ICBM2

---

## 📌 배경

웹 기술의 발전으로 누구나 브라우저에서 과학적 현상과 데이터를 시각화하고, 직접 실험하거나 탐구할 수 있는 시뮬레이션 프로젝트를 만들 수 있습니다. 본 문서는 Canvas, WebGL, 오픈소스 물리엔진 등 무료 기술을 활용해 GitHub+Vercel에 배포 가능한 신선한 과학/시뮬레이션 웹 프로젝트 아이디어를 정리한 것입니다. 기존에 많이 구현된 주제(3D 지구본, 게임, AI 에이전트 등)와 아래 참고 저장소의 아이디어는 제외했습니다.

## 참고: 기존 관련 저장소 (중복 금지)

- `terrarium` — 3D 방치형 생태계 육성
- `predator` — 생물 흡수 성장 아케이드
