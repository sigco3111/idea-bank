## 📌 배경

생산성과 일상 효율을 높여주는 웹 기반 유틸리티 프로젝트 아이디어를 찾는 개발자를 위해, 기존에 많이 구현된 뽀모도로, 투두, 메모, 계산기 등은 제외하고, 신선하면서도 실용적인 25가지 아이디어를 엄선했습니다. 모두 무료 기술 스택(React, Next.js, IndexedDB, PWA 등)으로 GitHub + Vercel 배포가 가능한 프로젝트입니다.
