> CesiumJS / Globe.gl / Three.js 기반 GitHub + Vercel 배포 가능 프로젝트 아이디어

## 📌 배경

3D 지구 맵 라이브러리를 활용해 GitHub에 저장소를 만들고 Vercel로 배포할 재미있거나 유용한 프로젝트 아이디어를 브레인스토밍한 결과물입니다. 비용 발생이 없고, 기존 저장소(sigco3111)에 없는 내용으로 제한했습니다.

### 참고: 기존 관련 저장소
- `superpower-web` — 193개 국가 정치/경제/군사 실시간 전략 시뮬레이션 게임
- `skill-worldmap` — 게임 오픈월드 스타일 인터랙티브 스킬 시각화
- `satellite-board` — 위성 궤도 3D 시각화 대시보드
- `world-data-map` — 국가별 통계 데이터 인터랙티브 지도
- `earthquake` — 전 세계 실시간 지진 추적 대시보드
- `geoPoly` — 지구마블 3D 보드 게임
- `iron-dome` — 3D 미사일 방어 시뮬레이션 게임
- `terrarium` — 3D 방치형 생태계 육성 시뮬레이션 게임
- `co2-board` — 국가별 CO₂ 배출량 비교 분석 대시보드
- `world-bank-board` — 세계은행 데이터 AI 비교 분석 대시보드

## PART 1: 일반 프로젝트 아이디어 (19개)

---

### 1. 🌐 cable-globe — 해저 인터넷 케이블 지도

**기술:** Globe.gl + Three.js  
**데이터:** Submarine Cable Map (공개 데이터), TeleGeography

**내용:**
- 전 세계 해저 광케이블 경로를 3D 아크로 시각화
- 케이블 클릭 시 → 대역폭, 착지국, 설립연도, 소유사 정보
- 랜딩 포인트에서 펄스 애니메이션 (데이터 흐름 느낌)
- "이 케이블이 끊기면 어떤 국가가 영향받을까?" 시뮬레이션 모드

**포인트:** 인터넷의 물리적 실체를 보여주는 건 흔하지 않고 시각적으로 아름다움

---

### 2. ✈️ sky-radar — 실시간 항공기 추적 글로브

**기술:** Globe.gl + OpenSky Network API (무료)

**내용:**
- 실시간 비행기 위치를 글로브 위에 점으로 표시
- 출발→도착 아크 경로 애니메이션
- 클릭 시 항공편번호, 고도, 속도, 항공사 정보
- 공항별 통계 (출발/도착 빈도 히트맵)

**포인트:** OpenSky API는 완전 무료. 실시간 데이터라 볼 때마다 다름

---

### 3. 💣 nuke-globe — 핵실험 역사 타임라인

**기술:** CesiumJS

**내용:**
- 1945~현재까지 전 세계 핵실험 위치를 시간순으로 재생
- 국가별 색상 구분, 폭발력 크기 = 점 크기
- 재생 중 연도 슬라이더로 탐색
- 클릭 시 실험명, 폭발력(킬로톤), 핵종류(수소/원자), 관련 역사적 배경

**포인트:** CTBTO 공개 데이터 활용. 교육적 가치가 높고 시각적 임팩트가 강함

---

### 4. 🏛️ empire-timeline — 역사 제국 영토 변천 지도

**기술:** CesiumJS + 커스텀 폴리곤 렌더링

**내용:**
- 로마, 몽골, 오스만, 대영제국 등 주요 제국의 영토를 시간축으로 재생
- 슬라이더로 연도 이동 시 국경이 부드럽게 변화
- 영토 넓이 그래프 하단 표시
- "이 시기에 동시대에 존재하던 제국들" 비교 모드

**포인트:** 역사 교육용으로 강력. Cesium의 폴리곤 + 타임라인 기능에 딱 맞음

---

### 5. 🌊 tsunami-sim — 쓰나미 파동 전파 시뮬레이터

**기술:** Globe.gl + WebGL Shader

**내용:**
- 역사적 쓰나미(2004 인도양, 2011 도호쿠 등) 발생 지점에서 파동 확산 시뮬레이션
- 파고/도달 시간/피해 규모를 인터랙티브하게 표시
- 해안선 근처에서 파도 높이 변화 시각화
- 사용자가 직접 지점 클릭해서 "가상 쓰나미" 발생시켜볼 수도

**포인트:** 게임적 요소(직접 테스트) + 교육적 가치. 시각적으로 멋짐

---

### 6. 🦋 migration-globe — 동물 이동 경로 실시간 시각화

**기술:** Globe.gl + Movebank 공개 데이터

**내용:**
- 고래, 독수리, 바다거북, 군집새 등 동물들의 GPS 추적 경로
- 종별 색상/속도/패턴 차이 시각화
- 재생 버튼으로 1년 주기 이동 경로 따라가기
- 클릭 시 개체 정보 (이름, 종, 나이, 이동 거리 누적)

**포인트:** Movebank API 무료. "지구 위를 여행하는 생명들"이라는 테마가 감성적

---

### 7. 📡 satellite-live — 위성이 지금 당장 뭘 보고 있나

**기술:** CesiumJS + CelesTrak TLE 데이터 (무료)

**내용:**
- 궤도 위성 실시간 위치 + 각 위성의 지상 커버리지 원 표시
- Starlink, ISS, KH-11 등 카테고리별 필터
- 특정 위성 선택 시 "지금 이 위성이 촬영하는 지역" 하이라이트
- 시간 배속으로 위성 궤도 예측

**포인트:** satellite-board가 궤도 시각화였다면, 이건 "지상 피사체" 관점. 완전히 다른 접근

---

### 8. 🌋 volcano-pulse — 화산 활동 글로벌 모니터

**기술:** Globe.gl + USGS Volcano API (무료)

**내용:**
- 활화산 위치를 점으로 표시, 활동 수준에 따라 크기/색상 변화
- 과거 대분출(베수비오, 크라카타우, 핀아투보) 타임라인 재생
- 분출 시 화산재 확산 방향/범위 애니메이션
- "가장 위험한 화산 Top 10" 랭킹 + 관련 마을/인구 데이터

**포인트:** earthquake(지진)와 달리 화산 전용은 없음. USGS 데이터 완전 무료

---

### 9. 🗣️ lang-globe — 전 세계 언어 지도 + 소리 듣기

**기술:** Globe.gl + Web Audio API

**내용:**
- 국가 클릭 시 해당 국가 공용어/방언 목록 표시
- TTS(Web Speech API)로 실제 발음 들려주기 ("안녕"을 193개국 언어로)
- 언어 계통수(language family)를 색상으로 구분
- "이 지역에서만 쓰이는 멸종위기 언어" 하이라이트

**포인트:** 교육적이고 인터랙티브. "언어 다양성 지도"라는 컨셉이 흔치 않음

---

### 10. 🏴‍☠️ pirate-ship — 역사적 해적 항로 추적 게임

**기술:** Globe.gl + Three.js

**내용:**
- 흑표범, 드레이크, 정일이 등 유명 해적/탐험가의 항로 재현
- 재생 모드로 항로 따라가며 각 항로 포인트에서 역사적 사건 팝업
- "해적 모드" — 사용자가 직접 항로를 그어서 목적지까지 도달하는 미니 게임
- 바람/해류 패턴을 배경에 표시하여 항해 난이도 반영

**포인트:** 게임 요소가 있으면서 역사 교육. 기존 geoPoly가 보드게임이었으니 이건 "항해 게임"

---

### 11. 🏗️ wonder-build — 세계 불가사의 건설 타임라인

**기술:** CesiumJS + 폴리곤 3D 모델

**내용:**
- 피라미드, 만리장성, 콜로세움, 타지마할 등을 3D 지도 위에 배치
- 타임라인 슬라이더로 "건설 연도" 이동 시 해당 시대의 불가사의만 표시
- 건설 규모, 인력, 공사 기간 비교 차트
- "이 건축물을 짓는 데 사용된 돌의 양 = 몇 km 도로?"

**포인트:** 건축/역사 관심사 타겟. 시각적 완성도가 높음

### 12. 🐋 deep-ocean — 심해 탐험 지도

**기술:** Globe.gl + 해저 지형 데이터 (GEBCO 무료)

**내용:**
- 해저 지형 3D 렌더링 (해구, 수중산맥, 심해평원)
- 타이타닉 침몰 위치, 마리아나 해구, 대서양 중앙해령 등 주요 지점 마커
- 깊이 슬라이더로 "수심 -1000m", "-5000m" 등 레이어별 탐색
- 각 지점에서 발견된 심해 생물 이미지/정보 팝업

**포인트:** "바다 밑 지도"는 육지 지도에 비해 희소. 시각적으로 독특

### 13. 🚀 rocket-trace — 우주발사체 발사/추락 궤적 시각화

**기술:** CesiumJS

**내용:**
- SpaceX, NASA, ROSCOSMOS 등 발사체 궤적을 3D로 재생
- 1단/2단 분리 지점, 부스터 추락 지역 표시
- 발사장 위치 + 회수지(드론쉽, 육상) 연결 아크
- 발사 성공/실패 이력, 연도별 발사 횟수 차트

**포인트:** 우주 덕후 타겟. 공개 궤도 데이터 활용. iron-dome이 미사일 방어였으니 이건 "발사" 관점

### 14. 🌊 current-globe — 해양 순환/해류 실시간 시각화

**기술:** Globe.gl + WebGL Particle System

**내용:**
- 멕시코 만류, 북태평양 순환, 남극 순환 등 주요 해류를 파티클 애니메이션으로 표시
- 온도/속도에 따라 색상 변화 (차가운 파랑 → 따뜻한 빨강)
- "플라스틱 쓰레기가 어디로 흘러가는가" 모드
- 엘니뇨/라니냐 발생 시 해류 변화 비교

**포인트:** 파티클 시스템으로 아름다운 시각화 가능. tsunami-sim과 달리 "지속적인 흐름"에 초점

### 15. 🏟️ stadium-globe — 전 세계 스타디움 3D 투어

**기술:** Globe.gl + CesiumJS 3D Tiles

**포인트:** Cesium의 3D Tiles를 활용하면 실제 건물 모델 로딩 가능. 스포츠 팬 타겟

### 16. 📻 signal-globe — 전파/통신 도달 범위 시각화

**기술:** Globe.gl

**내용:**
- 주요 라디오/TV 방송국, 5G 기지국의 전파 도달 반경 원 표시
- 지형에 따른 전파 차단 시뮬레이션 (산 뒤쪽은 수신 불가)

- "이 지역은 통신 블랙홀" 하이라이트 (인구 밀집도 vs 통신 커버리지)

**포인트:** "보이지 않는 전파를 본다"는 컨셉이 흥미로움. 라디오 관심사 타겟

### 17. 🦠 outbreak-trace — 전염병 확산 시뮬레이터

- 일자별 확진자 수 파동 애니메이션
- "감염원을 여기로 바꾸면 어떻게 퍼질까?" 가상 시뮬레이션 모드

**포인트:** WHO 공개 데이터 + 공항 데이터 결합. 교육적 가치 높음

### 18. 🌌 dark-sky — 빛공해/밤하늘 품질 지도

**기술:** Globe.gl + NASA Black Marble 위성 데이터 (무료)

**내용:**
- 야간 위성 이미지를 글로브 텍스처로 렌더링 (밤의 지구)

- 도시 성장에 따른 빛공해 변화 타임라인 (1992~현재)

**포인트:** NASA Black Marble 데이터 무료. "밤하늘의 지도"라는 테마가 감성적이고 흔치 않음

- 극지방 오로라 발생 지역과 자기장 관계 시각화

**포인트:** 과학 교육용. 지구 내부를 "보여주는" 독특한 접근

### 1. 🏙️ city-tycoon — 도시 건설 경영 시뮬레이션

**기술:** Globe.gl + Three.js  
**난이도:** 높음

**게임 루프:** 글로브에서 빈 땅 클릭 → 도시 건설 시작 → 성장 → 랭킹 경쟁

**내용:**
- 주거/상업/공업 구역 배치, 인구 증가에 따라 성장
- 실제 지형·기후 반영 (사막엔 농업 불리, 해안가엔 무역 보너스)
- 도시 간 무역, 전쟁, 외교로 다른 플레이어 도시와 상호작용
- **승리:** 20턴 후 GDP + 영토 + 외교 종합 점수

**포인트:** SimCity + 글로브. superpower-web이 국가 단위였다면 이건 도시 단위

---

### 2. 🦠 bio-plague — 바이러스 확산 vs 방어 대전 게임

**기술:** Globe.gl  
**난이도:** 높음
