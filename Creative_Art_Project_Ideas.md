# PART 1: 드로잉/페인팅 도구

### 1. 🎨 라이브 브러시 실험실 — 실시간 브러시 커스터마이즈 드로잉
**기술:** Canvas API, WebGL  
**난이도:** 보통

**내용:**
- 다양한 브러시(질감, 크기, 투명도) 실시간 조절
- 사용자 정의 브러시 저장 및 불러오기
- 드로잉 결과 PNG/JPEG로 저장

**포인트:** 브러시 커스터마이즈와 실시간 미리보기 제공

### 2. 🖐️ 손글씨 모션 캡처 — 웹캠 손동작으로 드로잉
**기술:** Canvas API, PoseNet  
**난이도:** 보통

**내용:**
- 웹캠으로 손 움직임 추적
- 손가락 움직임으로 캔버스에 그림 그리기
- 다양한 색상/굵기 선택

**포인트:** 마우스 없이 손동작만으로 그림을 그릴 수 있음

### 3. 🖌️ 레이어 기반 페인팅 — 포토샵 느낌의 레이어 페인팅
**기술:** Canvas API  
**난이도:** 쉬움

**내용:**
- 여러 레이어 추가/삭제/이동
- 각 레이어별 투명도 조절
- 레이어 병합 및 내보내기

**포인트:** 웹에서 간단하게 레이어 기반 드로잉 경험 제공

### 4. 🖼️ AI 스타일 트랜스퍼 페인터 — 사진→화풍 변환 드로잉
**기술:** TensorFlow.js, Canvas API  
**난이도:** 어려움

**내용:**
- 사진 업로드 후 다양한 화풍(고흐, 모네 등) 적용
- 스타일 변환 후 직접 드로잉 추가 가능
- 결과 이미지 다운로드

**포인트:** AI 스타일 트랜스퍼와 드로잉의 결합

### 5. 🎶 오디오 반응 드로잉 — 음악에 반응하는 그림
**기술:** Web Audio API, Canvas API  
**난이도:** 보통

**내용:**
- 음악/음성 입력에 따라 브러시, 색상, 패턴 변화
- 실시간 시각화와 드로잉 결합
- 오디오와 드로잉 결과 동시 저장

**포인트:** 오디오와 드로잉의 실시간 상호작용
