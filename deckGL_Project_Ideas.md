# 🗺️ deck.gl 데이터 지도 시각화 프로젝트 아이디어 모음
> deck.gl / MapLibre / React 기반 GitHub + Vercel 배포 가능 프로젝트 아이디어  
> 생성일: 2026-04-28 | 작성: ICBM2

---

## 📌 배경

deck.gl은 Uber가 개발한 GPU 가속 대규모 데이터 시각화 프레임워크입니다. WebGL2 기반으로 수백만 개의 데이터 포인트를 부드럽게 렌더링하며, ArcLayer, HexagonLayer, ScatterplotLayer, TripsLayer 등 60+ 레이어를 제공합니다. MapLibre, Google Maps, Mapbox 등과 통합 가능합니다. 본 문서는 deck.gl을 활용해 GitHub+Vercel에 배포 가능한 창의적인 지도 기반 데이터 시각화 프로젝트 아이디어를 정리합니다.

---

## 참고: 기존 관련 저장소 (중복 금지)
- `world-data-map` — 국가별 통계 데이터 인터랙티브 지도
- `satellite-board` — 위성 궤도 3D 시각화 대시보드
- `earthquake` — 전 세계 실시간 지진 추적 대시보드
- `co2-board` — 국가별 CO₂ 배출량 비교 분석 대시보드
- `world-bank-board` — 세계은행 데이터 AI 비교 분석 대시보드
