# 🎮 웹 게임 프로젝트 아이디어 모음

> HTML/CSS/JS, React, Canvas, WebGL 기반 GitHub + Vercel 배포 가능 프로젝트 아이디어  
> 생성일: 2026-04-28 | 작성: ICBM2

---

## 📌 배경

이 문서는 웹에서 바로 플레이 가능한 창의적이고 다양한 장르의 게임 아이디어를 모았습니다.  
기본 기술 스택은 HTML, CSS, JavaScript, React, Canvas, WebGL 등이며, GitHub와 Vercel을 활용해 누구나 쉽게 배포할 수 있습니다.  
아래 아이디어들은 기존에 많이 구현된 2048, 테트리스, 월드맵, 카드게임, 경영 시뮬레이션, RPG 등과 겹치지 않는 새로운 장르와 기획을 중심으로 선정했습니다.

### 참고: 기존 관련 저장소
- `2048` — 클래식 2048 퍼즐 게임
- `candy-crush-saga` — 캔디 크러쉬 매치-3 퍼즐
- `wordle` — 단어 추측 퍼즐
- `poker-king` — 텍사스 홀덤 포커
- `Idle_RPG` — 방치형 던전 RPG
- `Kingdoms3Lite` — 삼국지 턴제 전략
- `convenience24` — 편의점 경영 시뮬레이션
- `stock-game` — 주식 시뮬레이션
- `predator` — 생물 흡수 성장 아케이드
- `diablock` — 디아블로 스타일 자동 전투
